什么是 EntityFramework

EntityFramework 是一款 ORM 框架(对象关系映射 Object Relational Mapping，简称 ORM），可以使.NET 开发人员能够使用特定领域的关系型数据。开发人员无需像往常一样编写大量的数据访问代码。

  • 设计数据库

  • 使用 client 程序连接数据库
  • 编写 SQL 进行数据库操作

这就是 ORM 框架的优点:屏蔽了繁复的数据库操作细节,让用户专注于业务模型的建立
